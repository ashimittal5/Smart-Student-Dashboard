# Smart-Student-Dashboard
## 📌 Project Overview
The **Smart Student Dashboard UI** is a modern and user-friendly interface designed to help students manage their academic activities efficiently. It provides a centralized platform where students can track attendance, assignments, performance, and important notifications.

This project is designed using **Figma** and focuses on creating an intuitive and visually appealing user experience.

---

## 🎯 Objectives
- To design a clean and interactive student dashboard  
- To provide a centralized academic management system  
- To improve usability and accessibility  
- To enhance student productivity  

---

## ✨ Features
- 📊 Dashboard Overview (attendance, assignments, performance)  
- 📅 Assignment Management (pending & completed tasks)  
- 📈 Performance Analytics (charts & graphs)  
- 🔔 Notifications (alerts & updates)  
- 👤 Student Profile
